Here is my basic site tutorial with github site creation and atom integration
  1. Download and install atom for windows (or mac, this tutorial focuses on windows)
  2. Download and install git, I recommend git from command prompt, instead of bash
  3. Register for github.com with student email, I recommend a short username, if possible
  4. Create a new repo with the exact name as your username, plus github.io.. username.github.io
  5. Press WindowsKey+R on windows and type "cmd" then enter (to open the command prompt)
  6. type: git config --global user.name "(YourGithubUsername)"
  7. For github to function, the files in a cloud repository are replicated to your local machine, and update are typically made manually.
    You will need to relaunch atom
  8. In atom, press ctrl+shift+8 (to open the github tab)
